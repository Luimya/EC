# YAMIICHI Website UI Kit

Interactive click-through prototype of the YAMIICHI e-commerce website.

## Screens
1. **Homepage** — Hero + featured categories + new arrivals
2. **Shop** — Product grid with filters
3. **Product Detail** — Product page with size selector, images, description
4. **Cart / Checkout** — Cart summary + checkout form
5. **About** — Brand story page

## Usage
Open `index.html` for the full interactive prototype. Navigate using the top nav or in-page CTAs.

## Components
- `Nav` — sticky top nav with logo, search, cart icon
- `Hero` — full-bleed gradient hero with headline + CTA
- `ProductCard` — product thumbnail, badge, name, price
- `ProductGrid` — responsive 4-col grid with filter sidebar
- `ProductDetail` — image carousel placeholder + variant selector
- `CartItem` — cart row component
- `CheckoutForm` — address + payment fields
- `Footer` — links + brand tagline

## Design Tokens
All tokens are defined in `../../colors_and_type.css`.
Fonts: Noto Sans JP (Google Fonts), Space Grotesk (Google Fonts).
